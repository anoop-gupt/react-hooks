Basic hooks

useState
Simplest type. It basically initially sets a state variable, returns this and also contains a method for updating. const [count, setCount] = useState(initialCount); setCount(prevCount => prevCount + 1)} 
useEffect

useContext
Allows for using existing consumers and providers in the context api const value = useContext(MyContext);

Can I declare my own? Why, yes you can üòä

  • Convention: Always start the hooks function with use => useMyCoolState()
  • React docs
  • List of hooks

In essence..

  • Similar to how it works for classes
  • Can have state in functions
  • Can make your own hooks - to be shared between components
  • Only define hooks on top level of functions
  • Only works in functional components
  • Don't use hooks outside of React...
  • ..also; state is not shared between hooks of the same kind - as each call to them will be isolated üòä
